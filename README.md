REQUISITOS
----------
- Python 3.x (cualquier versión 3.6 o superior)
- No requiere librerías externas, solo módulos estándar de Python

ARCHIVOS
--------
- etapa2.py   Código de la Etapa 2 (Parte A y Parte B)

EJECUCIÓN
---------
1. Abrir una terminal en la carpeta donde está el archivo
2. Ejecutar el siguiente comando:
   python etapa2.py

SALIDA ESPERADA
---------------
El programa imprime en consola:

PARTE A — Cifrado de flujo con XOR
  - Mensaje original
  - Mensaje cifrado en hexadecimal
  - Mensaje recuperado (debe ser igual al original)

PARTE B — Cifrado por bloques
  - Mensaje original y clave
  - Transformaciones por bloque (sustitución, transposición, XOR)
  - Mensaje cifrado en hexadecimal
  - Mensaje recuperado (debe ser igual al original)
